You have probably read a bit about Hungry-For-Blocks and now want to know how to get it. There are several ways to get it, all of which are straight forward and user friendly.

Note:All the following instructions are written for google chrome and window 10, sorry if you use another browser or operating system.

The first way is by using the bookmark.

  1. If you do not have the bookmarks bar shown (the bar with the links in it) press Ctrl + Shift + B It should show a bar below the adress bar.
  2. Drag the folowing link onto your bookmarks bar: Hungry-For-Blocks
  3. Go to scratch and open the project that you want to use Hungry-For-Blocks on.
  4. Click the bookmark.
  5. If you look in the editor the blocks should now be in the more blocks tab.

The second way is by using the userscript.

A userscript is something that runs a script every time you go into a webpage.

  1. To run the userscript you will need a userscript manager. I recomend tampermonkey. Click here to get to its page and then click install.
  2. Then a dialouge should open with the option to add the extension. Click the add button and a short while later a new icon should show up at the top left of your screen. This means that it was succesful.
  3. Click the following link to install Hungry-For-BlocksInstall
  4. Now every time you go into scratch it will be loaded for you without you having to do anything.
